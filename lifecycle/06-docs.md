# 阶段六：文档与知识管理

## 阶段描述

技术文档编写和团队知识沉淀。AI 能大幅降低文档编写成本。

## 典型任务

| 任务 | 说明 |
|------|------|
| 接口文档 | 根据代码生成 API 文档 |
| 设计文档 | 撰写技术方案和设计文档 |
| 会议纪要 | 总结会议要点和待办 |
| 技术分享 | 准备分享材料 |

## 推荐 AI 工具

| 工具 | 用法 | 推荐度 |
|------|------|--------|
| Claude | 长文档撰写和组织 | ★★★★★ |
| Doc-coauthoring Skill | 结构化文档协同撰写 | ★★★★★ |
| Daily-report Skill | 自动生成工作日报 | ★★★★☆ |
| Diagram Skills | 生成架构图和流程图 | ★★★★☆ |

## 实用 Prompt 示例

### API 文档生成
```
请根据以下 Controller 代码生成 API 接口文档，格式为 Markdown 表格，包含：
- 接口路径和方法
- 请求参数说明
- 响应字段说明
- 错误码列表
- 调用示例

[粘贴 Controller 代码]
```
