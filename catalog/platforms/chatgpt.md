# ChatGPT

## 基本信息

| 字段 | 值 |
|------|-----|
| 开发商 | OpenAI |
| 当前版本 | GPT-4o / o1 / o3 |
| 访问方式 | 网页 / API / 移动端 |
| 适用阶段 | 全阶段通用 |
| 免费额度 | GPT-4o mini 免费，GPT-4o 有限额 |

## 核心能力

- 通用对话与问答
- 代码生成与审查
- 文档撰写与翻译
- 数据分析与可视化（Code Interpreter）
- 图片理解（多模态）

## 最佳实践场景

| 场景 | 用法 | 效果 |
|------|------|------|
| 需求分析 | 粘贴 PRD，让 GPT 梳理技术要点 | 快速提取关键信息 |
| SQL 优化 | 提供表结构和慢查询，请求优化建议 | 给出索引和改写建议 |
| 技术调研 | 对比多种技术方案的优劣 | 结构化对比分析 |

## 常用 Prompt

```
你是一名资深 Java 后端工程师，请帮我分析以下需求文档，提取技术要点和潜在风险点：
[粘贴需求内容]
```

## 局限性

- 训练数据有截止日期，最新技术信息可能不准确
- 长对话容易丢失上下文
- 代码生成需要仔细审查
