# Claude

## 基本信息

| 字段 | 值 |
|------|-----|
| 开发商 | Anthropic |
| 当前版本 | Claude 4 Opus / Sonnet |
| 访问方式 | 网页 / API |
| 适用阶段 | 需求分析、文档、方案设计、编码 |
| 核心优势 | 超长上下文（200K tokens）、代码质量高 |

## 核心能力

- 超长文档理解和分析（200K tokens 上下文窗口）
- 高质量代码生成，尤其擅长复杂逻辑
- 严谨的推理和分析能力
- Artifacts 功能支持实时预览代码/文档

## 最佳实践场景

| 场景 | 用法 | 效果 |
|------|------|------|
| 方案设计 | 提供完整需求，让 Claude 生成技术方案 | 方案完整度高 |
| 长文档分析 | 上传整个设计文档请求总结 | 利用长上下文优势 |
| 代码重构 | 提供大段代码请求重构建议 | 建议质量高且详细 |

## 常用 Prompt

```
请作为系统架构师，基于以下业务需求设计技术方案。要求包含：
1. 整体架构图描述
2. 核心模块划分
3. 数据库表设计
4. 接口定义
5. 关键技术选型理由

需求内容：[粘贴需求]
```

## 局限性

- API 调用成本较高
- 部分地区访问不便
- 实时信息获取能力有限
