# DeepSeek

## 基本信息

| 字段 | 值 |
|------|-----|
| 开发商 | DeepSeek（深度求索） |
| 当前版本 | DeepSeek-V3 / DeepSeek-R1 |
| 访问方式 | 网页 / API / 本地部署 |
| 适用阶段 | 编码、翻译、问答 |
| 核心优势 | 高性价比、开源可私有部署 |

## 核心能力

- 代码生成与补全（编码能力突出）
- 深度推理（R1 模型）
- 中英文翻译
- 数学推理和逻辑分析

## 最佳实践场景

| 场景 | 用法 | 效果 |
|------|------|------|
| 代码生成 | 描述业务逻辑让其生成 Java 代码 | 代码质量不错且成本低 |
| 技术问答 | 询问框架用法和最佳实践 | 回答准确度高 |
| 逻辑推理 | 使用 R1 模型分析复杂问题 | 推理链清晰 |

## 常用 Prompt

```
请用 Java + Spring Boot 实现以下功能，使用 MyBatis-Plus 作为 ORM 框架：
[描述业务需求]

```

## 局限性

- 模型能力整体略逊于 GPT-4o/Claude
- 多模态能力相对有限
