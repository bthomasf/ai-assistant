# GitHub Copilot

## 基本信息

| 字段 | 值 |
|------|-----|
| 开发商 | GitHub (Microsoft) |
| 适用 IDE | VS Code / JetBrains / Neovim |
| 适用阶段 | 编码、测试 |
| 核心优势 | 无缝嵌入 IDE，补全体验丝滑 |

## 核心能力

- 实时代码补全（Ghost Text）
- 基于注释生成代码
- 单元测试生成
- Copilot Chat 对话式编程

## 最佳实践场景

| 场景 | 用法 | 效果 |
|------|------|------|
| 样板代码 | 写好类名和注释，自动补全方法体 | CRUD 代码秒生成 |
| 单元测试 | 选中方法请求生成测试 | 自动覆盖主要分支 |
| 代码翻译 | 将 Python 脚本改写为 Java | 结构转换准确 |

## 局限性

- 长上下文理解不如独立 AI 平台
- 复杂业务逻辑生成质量参差不齐
- 需要付费订阅
