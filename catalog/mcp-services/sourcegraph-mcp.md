# Sourcegraph MCP

## 基本信息

| 字段 | 值 |
|------|-----|
| 类型 | MCP Server |
| 能力 | 全局代码搜索、跨仓库检索 |
| 适用阶段 | 编码、排查 |
| 核心优势 | 支持正则搜索和结构化查询 |

## 核心能力

- **跨仓库搜索**：一次搜索覆盖所有代码仓库
- **正则搜索**：支持复杂的正则表达式匹配
- **文件内容获取**：直接获取指定文件的内容
- **上下文定位**：搜索结果包含周围代码上下文

## 典型用法

| 场景 | 操作 | 效果 |
|------|------|------|
| 故障排查 | "搜索错误码 BIZ_10086 的定义位置" | 跨仓库定位 |
| 接口调用方 | "搜索哪些服务调用了 UserService.getById" | 评估影响面 |
| 配置排查 | "搜索包含特定配置项的文件" | 快速定位配置 |
