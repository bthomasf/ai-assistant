# MCP 服务类

Model Context Protocol 服务，为 AI 模型提供调用外部工具的能力。通过 MCP，AI 可以直接访问代码仓库、任务系统、日志平台等。

## 服务列表

| MCP 服务 | 能力 | 推荐指数 |
|----------|------|----------|
| [GitLab MCP](gitlab-mcp.md) | 代码搜索、仓库管理、MR 管理 | ★★★★★ |
| [Sourcegraph MCP](sourcegraph-mcp.md) | 全局代码搜索 | ★★★★☆ |
